## 为什么写博客？

在信息爆炸的时代，写博客似乎是一件「复古」的事情。但我相信，拥有一个属于自己的网络空间，记录思考、分享知识，是一件非常有价值的事情。

### 选择 Hexo + GitHub Pages

搭建这个博客，我选择了以下技术方案：

| 工具 | 用途 |
|------|------|
| [Hexo](https://hexo.io/) | 静态博客框架 |
| [Butterfly](https://github.com/jerryc127/hexo-theme-butterfly) | 博客主题 |
| [GitHub Pages](https://pages.github.com/) | 免费托管 |

**为什么选这套方案？**

- **完全免费**：GitHub Pages 提供免费的静态网站托管
- **高度自定义**：Hexo 有丰富的主题和插件生态
- **Markdown 写作**：用最简洁的方式记录想法
- **版本控制**：所有文章都在 Git 管理之下
- **自动部署**：通过 GitHub Actions 实现推送即部署

### 开始写作

- 🖥️ 技术学习笔记
- 🛠️ 项目实战经验
- 💡 灵感与思考
- 📝 生活记录

希望这个博客能成为我持续成长的见证。如果你偶然来到这里，希望能带给你一点启发。
